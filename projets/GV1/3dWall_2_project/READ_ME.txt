3d Wall Amiga Version

Written by Yawning Angel, 2024.

Please note all the relevant files for this game are contained in this folder.

To run it, load the 3d_wall_3.amos file into your copy of AMOS Professional.
NOTE: This is not a compiled game and needs to be run from within AMOS.

This game was written on a real Amiga A1200, with 2Mb Chip Ram and 64Mb expansion Ram.

Instructions on how to play the game can be seen in the accompanying video "Building a 3D Wall in AMOS"
available on my YouTube channel. See @YawningAngelRetro on YouTube.
